# Security Policy

## Supported Versions

| Version | Support          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.3.x   | :white_check_mark: |
| < v1.3  | :x:                |

## Reporting a Vulnerability

To report a vulnerability, create a new issue in the [Issues](https://github.com/prisect/dimanet/issues) tab.
